Navigate to the parent A2A folder

Open a terminal, activate the virtual env.

Run the following command to run the remote agent (prime_agent):

# adk api_server --a2a --port 8001 ./a2a_basic/remote_a2a


Open a new terminal, activate the virtual env.

run `adk web` command to run the local agent (root_agent)

# adk web
